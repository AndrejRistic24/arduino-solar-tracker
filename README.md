# ARDUINO SOLAR TRACKER  


Simple, 1 servo model for tracking the light (Sun) with a fixed elevation tailored after its location.  


## LIST OF COMPONENTS  

- 1x Arduino Board  
- 1x Solar Panel  
- 1x Servo Motor  
- 1x Breadboard  
- 1x USB-C Cable  
- 2x LDR Sensors
- Multiple  Wires, Resistors, LE Diodes...
